# 📊 PHASE 2: 완전 버전 개선 제안

**기한**: 반나절 (4-6시간)  
**목표**: Barobogi의 성장을 기록하는 완성도 높은 웹사이트로 진화

---

## 🎨 **1. 디자인 개선 (Design Enhancement)**

### 1.1 헤더 업그레이드
- [ ] 브랜드 로고/아이콘 추가 (예: 🌱 또는 커스텀 SVG)
- [ ] 다크모드 토글 버튼 (햇빛/달 아이콘)
- [ ] 헤더 배경 그래디언트 (선택사항: #5B8DEE → #7B9FFF)
- [ ] 로고 클릭 시 자연스러운 애니메이션

### 1.2 카드 & 레이아웃
- [ ] 카드에 미니 그래디언트 배경 추가 (white → #F9FAFB)
- [ ] 호버 시 transform 애니메이션 강화 (translateY, scale)
- [ ] 그림자 효과 업그레이드 (더 세련된 blur)
- [ ] 모바일에서 수직 카드 레이아웃

### 1.3 타이포그래피 개선
- [ ] 페이지 제목 크기 조정 (32px → 40px)
- [ ] 섹션 헤더 자간 조정 (letter-spacing 개선)
- [ ] 본문 가독성 높이기 (line-height 1.7 → 1.8)

### 1.4 다크모드 구현
- [ ] CSS 변수 활용: `--bg-primary`, `--text-primary` 등
- [ ] `prefers-color-scheme` 감지
- [ ] localStorage에 사용자 선택 저장
- [ ] 모든 페이지 대응 (index, logs, ai-study, stock-dashboard)
